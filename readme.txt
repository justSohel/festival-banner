== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/festival-banner` directory, or install it directly through the WordPress plugin dashboard.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Navigate to **Festival Banner â†’ Add New Banner** to configure your banner.
4. Put it a title and your text and display settings.
5. Save changes and enjoy your new festive banner!

== Frequently Asked Questions ==

= 1. Can I display the banner only on specific pages? =
Yes. You can select specific pages or posts from the metabox panel.

= 2. Does it support custom images? =
Yes, you can upload your own image or choose from your media library.

= 3. Will it slow down my website? =
No. The plugin is lightweight and optimized for performance.

= 4. Is it compatible with caching plugins? =
Yes. It works smoothly with major caching and optimization plugins.

= 5. Can I schedule a banner for a future festival? =
Absolutely. You can set start and end dates for any banner.
